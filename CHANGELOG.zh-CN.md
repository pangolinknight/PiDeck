# 更新日志

[English](CHANGELOG.md)

这里记录 pi-desktop 各版本的重要变化。

## v0.1.7 - 2026-06-01

### 优化
- 缩小项目列表默认宽度，为会话区域留出更多空间。
- 优化项目搜索框和新增按钮布局，窗口变窄时新增按钮仍保持可见。

## v0.1.6 - 2026-06-01

### 优化
- 优化会话消息中的 Markdown 表格渲染，补充边框、间距、表头样式，并让宽表格安全横向滚动。
- 将不明显的 textarea 原生右下角拉伸改为输入框顶部可见拖拽条。
- 输入框拉伸增加高度边界，避免输入区域展开后挤占过多会话阅读空间。

## v0.1.5 - 2026-06-01

### 修复
- 调整会话页顶部布局，让较长项目路径和会话操作按钮在窄窗口下更稳定显示。

## v0.1.4 - 2026-05-31

### 新增
- 新增运行中 Agent 的停止 / 中断按钮，底层使用 pi RPC `abort`。
- 新增模型首个流式 token 到来前的“正在响应”动画。
- 新增工具调用聚合卡片，避免一次问答中的多个 tool 消息占满对话区域。
- 工具调用组默认只展示少量摘要，支持展开查看完整详情。

### 优化
- 工具调用详情默认折叠，内容较长时独立滚动。
- 运行中和失败的工具调用有更清晰的视觉状态。

## v0.1.3 - 2026-05-31

### 新增
- 新增启动时 pi CLI 环境检测，并通过可见弹框展示检测结果。
- 新增通用 pi 命令定位器，适配打包后的 Electron 环境。
- 设置页新增手动检测 pi 环境入口。
- 设置页新增当前版本显示和“检测更新”入口，点击跳转 GitHub Releases。
- 新增静态启动页，减少渲染进程加载前的白屏感。

### 优化
- 打包应用在窗口准备好后再显示，减少启动白屏。
- 项目列表改为延后异步加载，让主界面更快显示。
- pi CLI 检测覆盖常见 PATH、npm、pnpm、Yarn、Volta、mise、nvm、asdf、bun、deno 和本地 bin 目录。
- Windows 下 `.cmd` 形式的 pi shim 使用 shell 检测，避免“实际能打开 Agent 但检测失败”的误报。
- 未检测到 pi CLI 时，引导用户打开官方安装文档。
- 历史会话支持从父目录启动但内容中引用子项目时，归类到对应子项目下。

## v0.1.2 - 2026-05-31

### 修复
- 修复 `.pi`、`.pi-desktop` 等隐藏目录的项目头像只显示点号的问题，现在会跳过前导点和空白。
- 将 `downloads/` 加入 `.gitignore`，避免本地下载产物进入发布版本。

## v0.1.1 - 2026-05-31

### 新增
- 新增 Electron Builder 打包配置，支持 Windows、macOS 和 Linux 目标。
- 新增目录构建和平台分发构建脚本。
- 新增打包应用图标资源。

### 优化
- 补充 Linux 包维护者信息。

## v0.1.0 - 2026-05-31

### 新增
- 初始版本 pi-desktop 工作台。
- 支持多项目桌面工作区，管理本地项目目录。
- 支持同时运行多个 pi RPC Agent。
- 支持历史会话抽屉和历史会话恢复。
- 支持文件抽屉、目录折叠和文件右键操作。
- 支持 Markdown 对话时间线和 Assistant 流式文本展示。
- 支持工具调用详情展示。
- 支持模型、思考等级、上下文和缓存状态显示。
- 支持 Git 分支展示和分支切换。
- 支持发送快捷键配置和面向桌面端的三栏布局。

### 修复
- 配置打包应用图标。
